# ADHD Focus Helper Landing

Static landing page for ADHD Focus Helper, ready for GitHub Pages.

## Files

- `index.html`: public landing page
- `privacy.html`: public privacy policy URL for Chrome Web Store
- `styles.css`: site styles
- `assets/`: local images copied from the extension store assets

## Local Preview

Open `index.html` directly in a browser, or run any static file server from this folder.

## GitHub Pages

Push this folder as a repository and enable GitHub Pages from the repository settings. Use `privacy.html` as the Chrome Web Store privacy policy URL.

## Chrome Web Store Link

The install buttons point to:
